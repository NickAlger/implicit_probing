API reference
=============

Every public object gets its own page (generated by :mod:`sphinx.ext.autosummary`). The **core** names
below are the flat public API, re-exported from ``implicit_probing`` itself::

    from implicit_probing import probe, ImplicitProblem, Multiset

The concrete-problem modules (reference, validation, and the FEniCS / JAX hooks) are imported on their
own line so their heavier dependencies stay optional.
